Kesef Kode Privacy Policy

Effective Date: May 10, 2026 · Applies to: Kesef Kode for macOS

Kesef Kode is a local-first AI IDE. This page documents our strict privacy architecture — what stays on your machine, what doesn't, and why.

1. On-Device AI Architecture

The AI coding engine ("Steve") runs the Qwen 3.5 Coder language model locally on your Mac via Apple MLX. All inference, code generation, code analysis, and prompt processing happens entirely on your Apple Silicon GPU and Neural Engine.

No tokens are streamed to, processed by, or stored on any remote server. Your source code is never uploaded for AI processing.

2. What Never Leaves Your Machine

📁 Source Code & Projects

All file reads, writes, AST analysis, and build operations are performed locally. Your project files are never transmitted.

🔑 API Keys & Secrets

Tokens, .env files, and credentials are processed in local memory only. Never sent to any server.

💬 AI Conversations

Chat history with Steve is stored on-device. Cloud sync is optional and user-initiated via your Kesef Plus account.

🧠 Model Weights & Cache

Downloaded model weights (~5.5 GB) are stored in your local HuggingFace cache. KV cache lives in RAM only.

3. Strictly Limited Network Access

Kesef Kode makes network requests only to the following destinations, and only when explicitly triggered:

DestinationPurposeWhen
huggingface.coOne-time model weight download (~5.5 GB)First launch / update
developer.apple.comBrowser Agent API documentation researchOn demand
stackoverflow.comBrowser Agent coding research fallbackOn demand
duckduckgo.comBrowser Agent web searchOn demand
kesefplus.appSparkle auto-update checks & downloadsPeriodic
supabase.coOptional account authentication & memory syncIf signed in

❌ No analytics SDKs · No tracking pixels · No crash telemetry to third parties.

4. Optional Account Data

If you sign in via Apple or Google, we receive your email, display name, and a unique identifier. We do not receive your password. Account data is stored on Supabase with Row-Level Security (RLS).

5. Data Footprint

The Qwen 3.5 model requires approximately 5.5 GB of disk space, cached at:

~/.cache/huggingface/hub/models--mlx-community--Qwen2.5-Coder-7B-Instruct-4bit/

You can delete this directory at any time to reclaim space. Steve will re-download on next launch. See the Data Management page or the in-app preferences for full uninstall instructions.

6. Your Rights

You have the right to access, correct, delete, and export your personal data. Contact ilan@kesefplus.app. We respond within 30 days.
