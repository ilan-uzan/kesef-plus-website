Getting Started

System Requirements

macOS 14 Sonoma or later. Apple Silicon (M1 or newer) required for local inference. 16GB RAM minimum, 32GB recommended for 13B models.

Installation

Download Kesef Kode from our TestFlight beta program. The app will automatically download and configure the default code model on first launch.

IDE Features

Full-featured code editor with syntax highlighting, intelligent autocomplete, and an agentic AI that understands your entire project context.

Browser Subagent

Kesef Kode includes a local browser agent that can research documentation, test APIs, and verify implementations — all without leaving the IDE.

Vision AI

Drop a UI mockup screenshot into the editor. The local Vision model translates it into SwiftUI code.

RAG System

Retrieval-Augmented Generation against 1M+ scraped Apple Developer documentation pages. Every code suggestion is grounded in real SDK documentation.

Xcode Integration

Kesef Kode generates .swift files that integrate directly into your Xcode project. Use both tools side by side for the optimal development workflow.

FAQ

Does Kesef Kode require internet?

No. All inference runs locally on your Mac's Apple Silicon. Internet is only needed for the initial model download.

What models are supported?

We ship a custom fine-tuned model optimized for Swift/SwiftUI. Support for custom models coming soon.
